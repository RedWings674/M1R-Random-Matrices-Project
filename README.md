# M1R-Random-Matrices-Project
# From Random Matrices to Graphs: A Spectral Analysis

An empirical and theoretical investigation into the eigenvalue statistics of Erdős-Rényi random graphs ($G(n,p)$) through modified adjacency matrices. This project explores the convergence of graph ensemble spectra to Wigner's Semicircle Law and identifies the critical density thresholds where this convergence breaks down, for a certain model value of n.

This repository contains the simulation and generation code for all plots used in the poster, as well as an appendix detailing the matrix scaling derivation
